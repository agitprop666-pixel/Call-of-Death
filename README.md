Call of Death is a video/audio/text
chat application. It is meant to be used with a VPN,
more specifically, something along the lines of
Zerotier or Tailscale. It's been fully tested with Zerotier.
www.zerotier.com
www.tailscale.com
It saves recorded video to your home directory (~) — so on Windows that's C:\Users\<username>\ and on Mac it's /Users/<username>/.
